# NovakDjokovic-GrandSlam-Analysis
• In this project I analyzed Novak Djokovic’s performance in Grand Slam tournaments, analysing key metrics such as total wins, aces, Grand Slam titles, average time on court, first serve points won and break points saved.

• It also includes a comparative analysis with his greatest rivals Rafael Nadal and Roger Federer to provide insights into Djokovic’s dominance in tennis.
